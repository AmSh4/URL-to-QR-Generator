# Biox Systems QR Code Generator

A Python-based utility to generate high-quality QR codes from URL addresses.

Prerequisites

Python 3.10 or higher

Pip (Python package installer)

Installation

Clone the repository:

git clone (https://github.com/navyavenkata1809-wq/Biox-QR-Generator)


Navigate to the project folder:

cd Biox-QR-Generator


Install dependencies:

pip install -r requirements.txt


Usage

Run the script using the following command:

python qr_generator.py


When prompted, enter the URL you wish to encode. The application will generate a file named biox_qr_output.png in the project directory.
